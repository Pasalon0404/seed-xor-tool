How this file encryption works

If you ever need to decrypt an .skb file manually or write your own decryption script in Python or Node.js, these are the exact cryptographic parameters used by this tool:

  • Algorithm: AES-256-GCM via the native Web Crypto API.
  • Key Derivation: PBKDF2 with a SHA-256 hash, 100,000 iterations, and a 16-byte random salt.
  • Initialization Vector (IV): 12-byte random IV.
  • Output Format: A standard JSON file containing the Base64-encoded salt, iv, and ciphertext.
